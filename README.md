# ENVS-193DS_final

# General information

This repository contains the final assignment for ENV S 193DS. The project includes statistical analysis and visualization of ecological datasets (Mountain Plover nest site occupancy, wetland flooding in the San Joaquin River Delta, and Santa Barbara giant kelp fronds data), as well as analysis and visualization of my personal screen time data. The assignment involves building statistical models, checking assumptions, creating data visualizations, and interpreting results.

This repository belongs to Sumaiya Alvi and was completed individually for the course.

# Data and file overview

The repository includes the following folders and files:

* `data/`: Contains the datasets used in the assignment, including the Mountain Plover dataset, the kelp dataset, the groundhog dataset, and the personal screen time dataset.  
* `code/`: Contains `final_assignment.qmd`, the Quarto file with all analysis, models, and visualizations, as well as the rendered HTML output.  
* `.gitignore`, `final_assignment_files`, `.Rhistory`, and this `README.md`.
